# Security Notes

This section contains my personal study notes about offensive security,
penetration testing concepts and common attack techniques.

The goal of these notes is to document important concepts, methodologies
and practical approaches used during security assessments and CTF challenges.

## Topics

### Brute Force
Techniques used to guess passwords or keys through repeated attempts.

### Hashes
Understanding hash functions, hash identification and password cracking techniques.

### OSINT / Information Gathering
Passive reconnaissance and public data collection about a target.

### Privilege Escalation
Techniques used to gain higher privileges after initial access.

### Pentest Methodology
Common methodologies used during penetration testing engagements.

## Purpose

These notes serve as a quick reference during:

- penetration testing labs
- Capture The Flag (CTF) challenges
- security research
- general cybersecurity studies
